# 🚦 Traffic Accident Hotspot Detection

A Machine Learning and Streamlit-based application for identifying accident-prone locations (hotspots) using historical traffic accident data. The project analyzes accident patterns, ranks high-risk areas, and visualizes them through an interactive dashboard and map.

---

## 📌 Features

- 📊 Interactive Dashboard
- 📍 Accident Hotspot Detection
- 🗺️ Interactive Hotspot Map
- 🏆 Top Accident Hotspots
- 🌍 Geographic Analysis
- 🔍 Data Filtering
- 📈 Accident Data Visualization

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/Traffic-Accident-Hotspot-Detection.git
```

Move into the project folder

```bash
cd Traffic-Accident-Hotspot-Detection
```

Install the required packages

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
